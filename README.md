Este repositorio contiene los notebooks y datasets de la parte práctica de la materia "Métodos para el Análisis de Datos I".

Estructura relevante:
- Unidad 1..5: carpetas por unidad con ejercicios y notebooks.
- Unidad 5: contiene un dataset sintético de ejemplo de "alfajores" y un notebook (U5_alfajores_analisis.ipynb) con análisis exploratorio, clustering (KMeans) y export de resultados.
